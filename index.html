Sudoku Assistant

I started this project because of leetcode 37 and because my father was working on sudoku puzzles from the newspaper. I largely figured out the recursive solver on my own, but I did get a bit of help from the solutions people posted on leetcode (e.g., I recognized that my code was doing extra recursive calls).

I then rewrote the solver in javascript so my father could use a web interface. This was my first attempt at javascript, and I got help from Google and Bard--especially for the input and output.

I went for a minimalist design and think it looks good and works well.
